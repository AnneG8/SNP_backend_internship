# SNP_backend_internship

Репозиторий с решениями задач для базовой стажировки по backend‑направлению для **SALT AND PEPPER**.

Проект представляет собой набор небольших учебных задач на Python с тестами и базовой инфраструктурой качества кода.

---

## Требования

Рекомендуемое окружение:

* **Python**: 3.10 или выше
* **Poetry**: версии ^1.8

Проект разрабатывался и тестировался именно с этими версиями.

---

## Установка зависимостей

1. Убедитесь, что Poetry установлен:

```bash
poetry --version
```

2. Установите зависимости проекта:

```bash
poetry install
```

---

## Активация виртуального окружения

Для входа в виртуальное окружение Poetry выполните:

```bash
poetry shell
```

Либо можно выполнять команды без активации окружения, используя `poetry run`, например:

```bash
poetry run pytest
```

---

## Структура проекта

* Каждая задача реализована в отдельном файле `task_XX.py`;
* Для **каждой задачи** написаны соответствующие тесты в папке `tests`.

---

## Запуск тестов

Для запуска всех тестов:

```bash
poetry run pytest
```

Или использовать `pre-commit` (инструкция ниже).

---

## Проверка качества кода

В проекте используются следующие инструменты:

* **pytest** — тестирование;
* **flake8** — стиль и ошибки;
* **mypy** — проверка типов;
* **isort** — сортировка импортов;
* **pre-commit** — единая точка запуска всех проверок.

### Запуск pre-commit

```bash
pre-commit run --all-files
```

#### Важно:

* **Активировать Poetry перед запуском `pre-commit` не требуется**
* `pre-commit` самостоятельно запускает все проверки внутри окружения Poetry
